# 小規模マネタイズ機能仕様書

## 🎯 概要

個人開発レベルで管理可能な、シンプルで実装しやすいマネタイズ機能の仕様書です。
クラシルなどの大手サービスと競合しない、ニッチな価値提供を目指します。

**収益モデル**: サブスクリプション + 広告収入のハイブリッド型

## 💎 プラン構成（シンプル2段階）

### 無料プラン
- 基本的な食材情報閲覧
- 楽天レシピ表示（広告付き）
- カテゴリ別食材一覧
- 基本的な調理法情報

### プレミアムプラン（月額200円）
- 広告非表示
- お気に入り機能（無制限保存）
- 簡易栄養計算機
- レシピのPDF保存
- 買い物リスト機能

## 🔧 実装予定機能（最小限）

### 1. お気に入り機能
```typescript
interface Favorite {
  userId: string;
  foodId: string;
  recipeId?: string;
  addedAt: Date;
  notes?: string; // 個人メモ
}
```

### 2. 簡易栄養計算機
- 選択した食材の合計カロリー計算
- 基本栄養素（タンパク質、炭水化物、脂質）の表示
- 1日の推奨摂取量との比較（年齢・性別ベース）

### 3. レシピPDF保存
- 楽天レシピの内容をPDF形式で保存
- 個人メモの追加機能
- オフライン閲覧対応

### 4. 買い物リスト機能
```typescript
interface ShoppingList {
  userId: string;
  items: {
    foodName: string;
    quantity: string;
    checked: boolean;
    addedFrom?: string; // レシピID
  }[];
  createdAt: Date;
  updatedAt: Date;
}
```

### 5. 広告非表示
- 楽天レシピ表示時の広告除去
- アプリ内バナー広告の非表示

## 📢 広告収入モデル（個人開発向け）

### 広告配置戦略
1. **バナー広告**
   - ページ上部・下部への配置
   - 食材詳細ページのサイドバー
   - 検索結果ページの間隔広告

2. **ネイティブ広告**
   - 関連食材の推奨枠
   - レシピ一覧内の自然な配置
   - 「おすすめ商品」として表示

3. **アフィリエイト広告**
   - 楽天市場の食材・調理器具
   - Amazon Fresh / 楽天西友ネットスーパー
   - 健康食品・サプリメント

### 広告プラットフォーム（実装しやすさ重視）

#### 1. Google AdSense
```typescript
interface AdSenseConfig {
  publisherId: string;
  adUnits: {
    banner: string;
    rectangle: string;
    native: string;
  };
  autoAds: boolean;
}
```
- **メリット**: 設定が簡単、自動最適化
- **想定収益**: 月間1,000PV → 500-1,500円

#### 2. 楽天アフィリエイト
```typescript
interface RakutenAffiliate {
  affiliateId: string;
  categories: string[]; // ['food', 'kitchen', 'health']
  commissionRate: number; // 2-8%
}
```
- **メリット**: 食材アプリとの親和性が高い
- **想定収益**: 月間10件成約 → 1,000-3,000円

#### 3. Amazonアソシエイト
- 調理器具・健康食品の紹介
- 月間5-15件成約で1,000-5,000円

### 広告表示ロジック
```typescript
interface AdDisplayRule {
  userType: 'free' | 'premium';
  pageType: 'home' | 'search' | 'detail' | 'recipe';
  adType: 'banner' | 'native' | 'affiliate';
  frequency: number; // 表示頻度
  conditions: {
    minPageViews?: number;

```

### 広告収益最適化

#### 1. ターゲティング戦略
- **食材カテゴリ別**: 野菜→有機野菜通販、肉→高級肉通販
- **健康志向**: サプリメント、オーガニック食品
- **調理レベル**: 初心者→簡単調理器具、上級者→専門器具

#### 2. 季節・イベント連動
- **春**: 新生活応援キッチン用品
- **夏**: 夏バテ対策食材・飲料
- **秋**: 旬の食材・保存食品
- **冬**: 鍋用品・温活食材

#### 3. ユーザー行動連動
- よく検索する食材の関連商品
- お気に入り食材の購入リンク
- レシピ材料の一括購入提案

## 🎯 収益目標（現実的）
